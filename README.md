## ğŸ‡ºğŸ‡¸ English

Project Elaheh is a sophisticated, two-server solution designed to bypass advanced internet censorship. It uses a **Foreign Upstream Server** to relay traffic for an **Iranian Edge Server**, creating a secure and resilient tunnel.

### System Requirements
*   **CPU:** 1 Core
*   **RAM:** 1GB
*   **Disk:** 2GB Free SSD
*   **OS:** Ubuntu 20.04+, Debian 11+, Rocky 9+

### Installation
Run the following command on both your Foreign and Iranian servers:
```bash
bash <(curl -Ls https://raw.githubusercontent.com/ehsanking/Elaheh-Project/main/install.sh)
```

### How It Works
The system requires two servers for maximum resilience:
1.  **Foreign Server (Upstream):**
    *   Install the script on a server **outside** of Iran (e.g., in Germany).
    *   During setup, select the **"Foreign Server"** role.
    *   After installation, log in to the simple dashboard and copy the **Connection Token**. This server acts as a secure exit point.
2.  **Iranian Server (Edge):**
    *   Install the script on a server **inside** Iran.
    *   During setup, select the **"Iran Server"** role.
    *   The setup wizard will ask for the **Connection Token** from your Foreign Server. Paste it in to link the two servers.
    *   This server provides the full management dashboard for creating and managing users.

**Security & Safety for Donors:**
*   **Encrypted Relay:** All traffic is end-to-end encrypted. You (the donor) cannot see the content of the traffic.
*   **No Logs:** The system is designed to forward packets without logging user activity, protecting your privacy.
*   **Reverse Tunneling:** The connection is initiated from the restricted side, making it harder for firewalls to detect and block your server.
